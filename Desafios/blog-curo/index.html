Semântica das Tags

No módulo 1 do curso aprendi sobre a semântica das tags, que basicamente é o significa que elas possuem!
Essa é uma inovação e uma das maiores mudanças do html5, cada tag tem um significado e não aparência, como
no caso das formatações de texto.

É importante lembrar que as tags fazem um grande diferencial no SEO do site.

Exemplo:

< strong >:
Negrigo e significa que a palavra merece uma atençao a mais, e não foi colocada apenas por elegância.(Semântica)
< em >:
Itálico e tem a mesma função do negrito, porém não tão chamativo.(Semântica)

CSS | font-weight: bold;
Tem a mesma aparência visual do < strong >, mas é so isso mesmo, porque para os motores de busca
essa palara não passa de uma palavra comum e não tem um significa escpecífico que mereça destaque.
CSS | font-style: italic
