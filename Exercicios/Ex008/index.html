Marcado

Podemos também criar um texto marcado usando a Tag < mark >

Grande e Pequeno

Hoje, pode demos usar a Tag < small > para deixar um texto pequeno.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele dever ser lido mas não considerado. Tag < del >

Texto Inserido

Podmos marcar um como inserido para dar uma enfazer e indicar que ele foi adicionado depois. Tag < ins >

Também existe o sublinhado com a Tag < u >

Texto Sobescrito e Subrescrito

Elevando um texto ex: 210. Tag < sup >

Abaixando um texto ex: H2O. Tag < sub >
